# Projeto Ranqueador de Livros 

Neste projeto será implementado um Web App para ranquear seus livros favoritos. Este permite adicionar livros com as respectivas notas e categorias, deletar os livros e atualizar as notas e categorias dos livros. Também será possível ver o ranking dos livros pelas notas ou pelas notas e uma categoria específica. 

## Funcionalidades 
- Adicionar livro com nota e categoria
- Ranquear livros pela nota
- Filtrar os livros pela categoria e apresentá-los em um ranking  
- Deletar livro que foi adicionado
- Atualizar nota ou categoria atribuída ao livro
