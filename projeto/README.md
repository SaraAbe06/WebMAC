# Projeto Ranqueador de Livros 

Neste projeto será implementado um Web App para ranquear seus livros favoritos. Este permite adicionar livros com as respectivas notas e autores, deletar os livros e atualizar as notas. Também será possível ver o ranking dos livros pelas notas, filtrando pelo nome do livro e/ou nome do autor. 

## Funcionalidades 
- Adicionar livro com nota e autor
- Ranquear livros pela nota
- Filtrar os livros pelo autor e/ou nome do livro e apresentá-los em um ranking  
- Deletar livro que foi adicionado
- Atualizar nota atribuída ao livro
